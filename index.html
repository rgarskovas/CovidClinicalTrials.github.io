Analyzing clinical trial growth for Covid19

The purpose of this project was to analyze the dynamics of clinical trials as they relate to Covid19.

To accomplish this analysis, we first used ClinicalTrials.gov API to assemble a dataset on over 2,700 trials.

After assembling the dataset, we used Matplotlib to plot various aspects of the trials. Factors we looked at included: dates, sponsors, sites, and enrollment. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
